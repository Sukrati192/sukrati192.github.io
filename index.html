Piece Of Work

The Battle of Neighborhoods of Delhi to start an Indian Restaurant

This project aims to find the best location to open an Indian restaurant in the city of Delhi, India, to maximize the profit of the owner. This project includes data extraction, analysis and k-means clustering. It provides best 10 locations to open an Indian restaurant in Delhi. Studies like this can help people to find the best neighborhoods to start a new business.

Sentiment Analysis

The aim of this project is to build a sentiment analysis model which allowed us to categorize words based on their sentiments, that is whether they are positive, negative, constraining, litigious,etc. and also the magnitude of it. In order to build my project, I made use of the tidytext package that comprises of sentiment lexicons that are present in the 'janeaustenR' dataset of ‘sentiments’. I used a lexical analyzer – ‘loughran’ in this project. Furthermore, I also represented the sentiment score through a plot and also made a visual report of wordcloud.

Color Detector

In this Python project, I built an application through which one can automatically get the name of the color by clicking on them. So for this, I used a data file that contains the color name and its values. Then I calculated the distance from each color and find the shortest one. This is used in numerous image editing and drawing apps.

Customer Segmentation
